如果源码引用了其它的第三方类库，那么使用
LOCAL_JAVA_LIBRARIES += *
来引用该库，注意千万不能是LOCAL_STATIC_JAVA_LIBRARIES,
这会引发一些错误。
另外请务必使用LOCAL_PROGUARD_ENABLED := disabled，否则可能无法编译通过
